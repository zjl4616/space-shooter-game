# 🎮 太空射击游戏 - 网页版

一个使用HTML5 Canvas和JavaScript开发的2D太空射击游戏，可以直接在浏览器中运行。

## 🌟 特性

### 🚀 游戏功能
- **玩家控制**：键盘控制飞船移动和射击
- **敌人系统**：自动生成，渐进难度
- **碰撞检测**：子弹vs敌人，玩家vs敌人
- **游戏机制**：分数系统，生命值管理
- **视觉效果**：动态星空背景，彩色UI

### 🎨 技术特点
- **纯前端**：无需安装，直接浏览器运行
- **响应式**：支持不同屏幕尺寸
- **跨平台**：Windows, macOS, Linux, 移动端
- **高性能**：使用Canvas渲染，流畅游戏体验

## 🚀 快速开始

### 方法1：直接打开HTML文件
1. 下载 `space_shooter_web.html`
2. 用浏览器打开该文件（双击或在浏览器中打开）

### 方法2：使用Python服务器
```bash
# 1. 确保有Python环境
python3 --version

# 2. 运行启动脚本
python3 start_web_game.py

# 3. 浏览器会自动打开游戏页面
```

### 方法3：使用Node.js服务器
```bash
# 1. 安装http-server（如果尚未安装）
npm install -g http-server

# 2. 启动服务器
http-server .

# 3. 浏览器访问 http://localhost:8080/space_shooter_web.html
```

## 🎮 游戏控制

| 按键 | 功能 | 说明 |
|------|------|------|
| **W / ↑** | 向上移动 | 飞船向上移动 |
| **A / ←** | 向左移动 | 飞船向左移动 |
| **S / ↓** | 向下移动 | 飞船向下移动 |
| **D / →** | 向右移动 | 飞船向右移动 |
| **空格键** | 发射子弹 | 向敌人射击 |
| **R键** | 重新开始 | 游戏结束后重新开始 |
| **ESC键** | 退出全屏 | 退出全屏模式 |

### 移动端支持
- **左侧触摸**：向左移动
- **中间触摸**：射击
- **右侧触摸**：向右移动

## 📊 游戏规则

### 得分系统
- ✅ **击中敌人**：+10分
- ❌ **敌人逃脱**：-5生命值
- 💥 **与敌人碰撞**：-20生命值
- 🎯 **游戏目标**：获得尽可能高的分数

### 难度递增
- 初始敌人生成频率：60帧/敌
- 每获得100分，关卡提升1级
- 每升一级，敌人生成频率降低5帧
- 最高难度：30帧/敌

## 🐛 故障排除

### 常见问题

**问题1：游戏无法加载**
- 确保浏览器支持HTML5 Canvas
- 检查控制台错误信息（F12 → Console）
- 尝试使用Chrome或Firefox

**问题2：键盘控制无效**
- 确保游戏窗口处于激活状态
- 检查键盘布局（可能需要切换输入法）
- 尝试使用备用按键（方向键代替WASD）

**问题3：游戏卡顿**
- 关闭其他标签页和程序
- 降低浏览器硬件加速设置
- 使用Chrome的性能模式

**问题4：触摸控制不灵敏**
- 确保触摸位置准确
- 尝试使用更大的触摸区域
- 建议在电脑上使用键盘游玩

### 调试工具
1. 按 **F12** 打开开发者工具
2. 查看 **Console** 标签获取错误信息
3. 使用 **Performance** 标签分析游戏性能
4. 使用 **Network** 标签检查资源加载
